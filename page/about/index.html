淡我前生孤寂梦，行云逝水纵吴歌。

1 minute read

Github: https://github.com/Zsbyqx20

在一个不知名的夜晚，终于开始着手准备这个主页；本来没有特别的想法来写这个 about ，但是竟也巧合遇到了点曲折，于是还是想来记录一下。

前日完成了在 Windows 系统上的工作，本来想转到 Linux 这边做一些文献整理和搭建网站内容的事情，重启之后但很神奇就进不了正常的 grub 界面了，再打开 Windows 后发现它自动给我装了一个不知名的更新 😶‍🌫️（从此再也不敢随便更新

于是和这个问题磕了很久。感觉国内的 Fedora 用户群体仍然是少数，大部分都是 Ubuntu 的资料；在 grub 启动的实现上 Fedora 和 Ubuntu 其实还是有区别的，所以并不能直接去用现成的答案，于是转而去科技服务队的群里求助，得到了一个 grub wiki 的网站 😅 那我还不如重装了哈哈哈

最后今天晚上终于用 live system 的方式给原先的系统重装了 grub，才终于能继续使用双系统，然后继续写这个主页博客；谨以此纪念我对操作系统的认识更深一层 🧐

whatever，欢迎到访这里。
